# Калькулятор

In-browser calculator with modern UI

## Описание

Современный калькулятор с красивым интерфейсом, работающий в браузере. Поддерживает основные математические операции, работу с клавиатуры и адаптивный дизайн для мобильных устройств.

## Функции

- ✅ Базовые математические операции (+, -, ×, /)
- ✅ Поддержка десятичных чисел
- ✅ Работа с клавиатуры
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Современный UI с анимированным градиентным фоном

## Деплой на Vercel

### Способ 1: Через Vercel CLI

1. Установите Vercel CLI (если еще не установлен):
```powershell
npm install -g vercel
```

2. Войдите в аккаунт Vercel:
```powershell
vercel login
```

3. Задеплойте проект:
```powershell
vercel
```

4. Для продакшн деплоя:
```powershell
vercel --prod
```

### Способ 2: Через GitHub

1. Создайте репозиторий на GitHub и загрузите код
2. Перейдите на [vercel.com](https://vercel.com)
3. Войдите через GitHub
4. Нажмите "Add New Project"
5. Выберите ваш репозиторий
6. Vercel автоматически определит настройки и задеплоит проект

### Способ 3: Через веб-интерфейс Vercel

1. Перейдите на [vercel.com](https://vercel.com)
2. Нажмите "Add New Project"
3. Импортируйте проект из Git репозитория или загрузите файлы напрямую
4. Vercel автоматически определит настройки из `vercel.json`

## Локальная разработка

Для запуска проекта локально:

```powershell
npm install
npm run dev
```

Или используйте любой статический сервер:

```powershell
npx serve .
```

## Использование

- **Мышь/Тач**: Нажимайте на кнопки калькулятора
- **Клавиатура**: Используйте цифры и операторы на клавиатуре
  - `Enter` или `=` - вычисление
  - `Escape` - очистка всего
  - `Backspace` - удаление последней цифры
